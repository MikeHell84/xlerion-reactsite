# CRM Admin (skeleton)

Este directorio contiene páginas administrativas placeholder para el CRM de Xlerion.

Páginas:
- `index.php` — Panel principal del CRM.
- `customers.php` — Clientes / Contactos.
- `leads.php` — Leads / Prospectos.
- `opportunities.php` — Oportunidades / Ventas.
- `activities.php` — Actividades y tareas.
- `communications.php` — Comunicaciones.
- `products.php` — Productos / Servicios.
- `invoices.php` — Cotizaciones / Facturación.
- `reports.php` — Reportes y métricas.
- `automations.php` — Automatizaciones.
- `settings.php` — Configuración del sistema.

API stubs:
- `/api/crm/auth.php`
- `/api/crm/customers.php`
- `/api/crm/leads.php`
- `/api/crm/opportunities.php`
- `/api/crm/activities.php`
- `/api/crm/communications.php`
- `/api/crm/products.php`
- `/api/crm/invoices.php`
- `/api/crm/reports.php`
- `/api/crm/automations.php`
- `/api/crm/settings.php`

Pruebas rápidas:
1. Abrir `public/admin/crm/index.php` en el navegador (vía servidor PHP).
2. Probar los endpoints JSON en `http://localhost:8000/api/crm/`.
